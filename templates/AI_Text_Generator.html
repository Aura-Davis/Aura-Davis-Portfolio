AI Text Generator

Github Link
Description

A Python project that uses Pandas and Tensorflow for machine learning to generate the next word based on a dataset forming a short paragraph.The AI is trained on 3,865 words 21,040 characters. It does 50 epochs over the sample data to create a relatively accurate text predictor. It then generates 50 follow up words to a start sentence.
